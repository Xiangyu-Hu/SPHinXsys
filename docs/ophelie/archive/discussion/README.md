# GPT discussion Archive

Historical ChatGPT discussion bundles for OPHELIE edge-flux, French reduced, TEAM7 L2/L3, and related validation rounds (June 2026).

## English-only policy

New and updated files under `docs/ophelie/` must be written in **English only**. 
Files in this directory are archived discussion records; they were translated to English during documentation consolidation.

## Layout

| File pattern | Content |
|--------------|---------|
| `OPHELIE_GPT_*` | French / edge-flux / complex edge-flux discussion packs |
| `TEAM7_*` | TEAM7 upload manifests and round summaries |
| `OPHELIE_STAGE2_*` | Stage 2 Cursor execution records |
| `OPHELIE_COMPLEX_*` | Complex edge-flux run records |
